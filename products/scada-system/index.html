SCADA System

SALCAN Process Technology have developed a Gas Field Supervisory Control & Data Acquisition application (SCADA) and have successfully implemented and proven this over multiple CSG / Gas Drainage wells. The application allows for remote monitoring and supervisory control of individual production wells and ancillary equipment located on the gas field.

Features of our SCADA system include:

  • Status of gas wells at a glance from overview page (Running/Stopped/In Alarm)
  • Ability to remotely start/stop production wells
  • Intuitive alarming interface for alarm annunciation and acknowledgement
  • Integrated data historian and trending
  • Integrated reporting for gas field and individual well heads (automated via email at user-defined frequency)
  • Event recording including recording of all operator actions
  • Implementation of various SCADA security levels based on job role
  • DNP3 communication protocol implementation meaning no loss of production data even when the communication network is down
  • Redundant servers (with automatic fail-over) providing increased reliability and performance

We can offer:

  • A SCADA application built upon the Schneider Electric ClearSCADA platform
  • A SCADA system based on a client's chosen platform
  • User Security groups – View Only, Operator, Supervisor, Engineer, Administrator
  • Historical data exported to PDF, Excel, other formats
  • Open capability to integrate with other systems
  • Alarm forwarding to email/SMS/pager system for site operators
